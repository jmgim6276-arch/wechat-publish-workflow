# Authoring Loop

This file defines how to convert captured material into a publishable公众号 draft.

## Principle

Do not write from abstraction first.

Write from:

1. scene
2. feeling
3. judgment
4. structure

## Conversion Flow

### 1. Extract the real sentence

Find the sentence that already contains the article's truth.

Examples:

- `我不是不喜欢技术了，我只是被错误的工作方式耗尽了`
- `我不是做不动了，我只是偏离了自己真正擅长的路径`

This sentence usually becomes:

- the title
- the pull quote
- or the final takeaway

### 2. Choose the article shape

Use one of these shapes:

1. reflection:
   - for心力、职业状态、长期感受
2. method:
   - for workflow, tools, repeatable practice
3. scene-to-insight:
   - for a concrete event that turns into a broader realization

### 3. Build 3 to 5 sections

Prefer:

1. opening scene or admission
2. what changed or what felt wrong
3. the user's current judgment
4. what this means for others
5. a quiet ending or question

Avoid making it too complete or too academic.

## Tone Rules

The target tone is:

- personal
- clear
- a little reflective
- not over-explained
- not too corporate

Cut:

- repeated explanation
- defensive clarification
- overly formal transitions

Keep:

- scene sentences
- judgment sentences
- memorable lines

## After Drafting

Once the structure is stable:

1. choose cover title candidates
2. choose summary
3. reduce density
4. enter style and publish flow
