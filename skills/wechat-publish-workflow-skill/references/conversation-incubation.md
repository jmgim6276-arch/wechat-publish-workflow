# Conversation Incubation

This file defines what to do when the user says something like `我要开始写公众号了`.

## Goal

Turn natural conversation into reusable writing material without making the user feel like they are filling out a form.

## Default Behavior

When the user enters capture mode:

1. listen for scenes
2. listen for emotional shifts
3. listen for repeated judgments
4. listen for one sentence that already sounds like a title or thesis

Do not rush into structure if the user is still unloading experience.

## What to Capture

Capture these four layers:

1. event:
   - what happened
   - what the user saw or experienced
2. feeling:
   - frustration, relief, confusion, excitement, shame, clarity
3. judgment:
   - what the user now believes because of the experience
4. transferable value:
   - what another person could recognize, learn, or borrow

## Agent Behavior

The agent should guide lightly:

- reflect back the user's core point
- tighten vague wording when needed
- mark possible article-worthy lines
- notice repeated themes across the conversation

Good prompts during capture:

- `你刚才那句其实很像这篇文章真正想说的话`
- `这一段更像经历，还是更像判断？`
- `如果只留一句最重要的话，会是哪一句？`

Avoid:

- full outlines too early
- formal interview tone
- forcing numbered frameworks before the material is mature

## Readiness Test

Material is mature enough when:

1. the user is no longer only venting
2. at least one judgment has stabilized
3. there is at least one concrete scene or example
4. there is a reason someone else would care

If not ready, keep collecting.

## Output of This Stage

Before drafting, the agent should be able to summarize:

- the core topic in one line
- the strongest title-like sentence
- the best opening scene
- the main takeaway for readers
